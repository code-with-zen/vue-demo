# Vue 3 Shopping Cart Demo 🛒

这是一个基于 **Vue 3 Composition API** 构建的电商购物车核心功能演示项目。旨在展示对 Vue 3 响应式系统及数据流的理解。

## 🛠 技术栈
- **Vue 3** (Script Setup)
- **Vite** (构建工具)
- **CSS3** (Flexbox 布局)

## ✨ 核心亮点
1. **响应式驱动**：使用 `ref` 实现数据驱动视图，无手动 DOM 操作。
2. **依赖追踪**：利用 `computed` 自动计算总价，利用 Vue 的缓存机制优化性能。
3. **组件化思维**：逻辑与视图分离，代码结构清晰。
